# 📋 书海回响 Landing Page 实施计划

## 1. 项目概述

| 项目 | 内容 |
|------|------|
| **项目名称** | 书海回响 (Echoes of the Book Sea) |
| **项目类型** | 单页 Landing Page |
| **技术栈** | Vite + React |
| **设计风格** | Notion Design System |
| **部署平台** | Vercel |

---

## 2. 页面结构

1. **Header** - 导航栏（Logo + 链接 + CTA）
2. **Hero** - 首屏（标题 + 副标题 + 按钮 + 装饰图）
3. **About** - 项目简介 + 核心理念
4. **Features** - 4功能卡片（月份牌/睡美人/主题卡/文学FM）
5. **Flow** - 数据处理流程图
6. **AIBotShowcase** - 展示前端介绍 + 截图
7. **BrowserExtension** - 浏览器插件介绍
8. **Footer** - GitHub/演示/联系方式

---

| AI Bot | 界面截图 | 800x500 |

### 图片优化策略
- 懒加载 + `loading="lazy"`
- 响应式 `srcset`
- 优先 WebP 格式
- 骨架屏占位

## 4. 设计规范（Notion Style）

- Primary: `#0075de` (Notion Blue)
- Background: `#ffffff` / `#f6f5f4` (warm white)
- Text: `rgba(0,0,0,0.95)`
- Border: `1px solid rgba(0,0,0,0.1)`
